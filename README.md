# Soli ሶሊ — My English Teacher

Interactive language learning for Ethiopian children. Game-driven. Audio-first. Voice-first.

## Deploy
1. Push to GitHub
2. Import on Vercel (Framework: Next.js)
3. Add `ANTHROPIC_API_KEY` for future AI conversation (optional)

## What Children Experience
Soli is an animated Ethiopian teacher who needs the child's help to learn English. The child helps Soli through adventures: popping word bubbles, shouting English words, building sentences, watering a garden of learned words, and teaching a younger student named Lemi.

## Local Dev
```bash
npm install
npm run dev
```
